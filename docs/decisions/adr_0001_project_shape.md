# ADR 0001：项目形态

## 状态

已接受。

## 背景

项目早期容易被理解为批量邮件报表工具，但当前目标是企业邮箱中的 AI 辅助窗口。

## 决策

项目第一阶段定位为“当前邮件 AI 辅助分析”。用户在企业邮箱中打开一封邮件后，点击按钮触发分析。系统展示摘要、优先级、分类、风险点、建议动作和回复草稿。

## 后果

- 不以批量报表作为核心产品形态。
- 不自动扫描所有邮件。
- 不自动发送、删除或归档邮件。
- 后端可保留 SQLite 记录用于调试和回看。
