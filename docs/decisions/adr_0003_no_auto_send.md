# ADR 0003：不自动发送邮件

## 状态

已接受。

## 背景

AI 生成内容可能存在误判、遗漏或不适合直接对外发送的表达。企业邮件还可能涉及价格、交期、付款、合同和法律责任。

## 决策

第一阶段所有 AI 生成回复只能作为草稿或建议，不允许自动发送邮件。

## 后果

- 回复草稿必须标记需要人工审核。
- 前端不能提供绕过人工确认的发送能力。
- 后端不能暴露自动发送接口。
- 涉及业务承诺的内容必须提示人工确认。
