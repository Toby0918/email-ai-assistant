# Triage 标签

工程技能使用五个规范 triage 角色。下表把这些角色映射到本仓库 GitHub Issues 使用的标签。

| `mattpocock/skills` 角色 | GitHub 标签 | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 维护者需要评估该 issue |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 规格完整，可交给自主 Agent |
| `ready-for-human` | `ready-for-human` | 需要人工实现或决策 |
| `wontfix` | `wontfix` | 不会继续处理 |

当技能提到某个 triage 角色时，使用对应的 GitHub 标签。若仓库以后改变标签命名，只修改本表右侧映射，避免创建重复标签。
