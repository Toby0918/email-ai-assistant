# 隐私规则

## 第一阶段原则

- 不接入真实邮箱账号，除非后续单独确认。
- 不读取真实邮箱数据。
- 测试邮件必须使用虚构或脱敏内容。
- 不提交真实邮件、数据库文件、日志中的敏感内容。

## 数据最小化

- 只处理当前打开邮件的必要字段。
- 不后台扫描邮箱。
- 不读取未授权邮件。
- 不采集与分析无关的个人信息。

## 展示规则

- 前端只展示当前分析所需内容。
- 错误提示不得包含密钥、token、完整堆栈或内部 prompt。
